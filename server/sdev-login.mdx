## Overview

The `sdev-login` library patches two well-known login service vulnerabilities in Shaiya Episode 6, providing essential security improvements for production deployments.

## Environment

- **Platform**: Windows 10
- **IDE**: Visual Studio 2019
- **Language**: C++ 20
- **Prerequisites**: [Microsoft Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x86.exe)

## Security Patches

This library addresses critical security vulnerabilities in the login service that could be exploited by malicious users.

<Warning>
The specific vulnerabilities are not disclosed in detail to prevent exploitation. Always use the patched version in production environments.
</Warning>

## Installation

1. Download the library from the repository
2. Ensure the Microsoft Visual C++ Redistributable is installed
3. Deploy the library to your login service directory
4. Restart the login service

<Note>
No additional configuration or database changes are required. The patches are applied automatically when the library is loaded.
</Note>
